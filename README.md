ipdata
======

### 欢迎使用IP查询

#### 方式一

**GET**

```
/api/ip/get/:ip.json
```
**Example**

```
http://ipdata.sturgeon.mopaas.com/api/ip/get/192.168.1.1.json
```


#### 方式二

**POST**

```
/api/ip/post.json
```

**json**

```
{
  "ip":"192.168.1.1"
}
```

**Example**

```
http://ipdata.sturgeon.mopaas.com/api/ip/post.json
```

_数据来自互联网,不能保证绝对的正确性_
